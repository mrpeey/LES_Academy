This repository contains solutions for the Week 4 AI assignment, including code, notebooks, and documentation.

## Contents

- `sort.py` — Sorting algorithms for lists of dictionaries (AI-powered and manual).
- `sort_comparison.ipynb` — Notebook comparing sorting methods.
- `login_test.py` — Selenium script for automated login testing.
- `breast_cancer_priority_prediction.ipynb` — Predictive analytics notebook using the Kaggle Breast Cancer Dataset.
- `Ethical_Reflection.txt` — Ethical reflection on model deployment and fairness.
- `Ethical_Reflection.pdf` — PDF version of the ethical reflection.
- `txt_to_pdf.py` — Script to convert text to PDF.
- `breast-cancer.csv` — Sample data (Kaggle Breast Cancer Dataset).
